## Getting Started

First, run the development server:

```bash

# or
yarn dev
# or
pnpm dev
# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Production Deploy Checklist (GitHub + Vercel)

1. Set your production site URL in Vercel:

- Environment Variable: `NEXT_PUBLIC_SITE_URL`
- Value example: `https://janak-engineers.vercel.app` (or your custom domain)

2. Validate locally before pushing:

```bash
npm run lint
npm run build
```

3. Push to GitHub:

```bash
git add .
git commit -m "Prepare site for production deploy"
git push origin master
```

4. In Vercel:

- Import/connect this GitHub repository.
- Keep Framework Preset as Next.js.
- Confirm `NEXT_PUBLIC_SITE_URL` is set.
- Deploy.
